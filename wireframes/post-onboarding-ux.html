Part 1A: Post-Checkout Interstitial

Shown once after first successful Stripe checkout. NOT shown if user cancels checkout (goes straight to Home with upgrade banner). Focuses on the moment, not payment methods.

Behavior One-time only (after ?checkout=success). "Deposit now" opens DepositSheet. "I'll do this later" goes to Home where fund-wallet banner catches them. Cancelled Stripe checkout skips this entirely, goes straight to Home. Stored in localStorage so it never shows again.
