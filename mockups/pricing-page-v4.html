What changed in v4
v4 responds to three things: (1) hide the trial so it doesn't cannibalise direct conversion, (2) differentiate Premium visually so the upsell is obvious, (3) tighten Screen 2 so users actually read it. Detailed trust scaffolding moves to the trial-start email.
Trial is now gated. Screen 1 no longer announces the trial. The only path is a small underlined link reading "I'm not ready to subscribe yet" below the plans. High-intent users don't see trial messaging and subscribe directly. Hesitant users self-select into the trial path by tapping the link.
Standard trimmed. Dropped leverage (not a differentiator Sarah read) and "full daily brief" (available to everyone). Now three features: asset count, manual approval, no fees. Every item is something a prospect is actually deciding about.
Premium differentiated visually. Deeper shadow (6px vs none on Standard), 3px border vs 2px, signal-green tint background, signal-green "Recommended" badge with ink border. It reads as a noticeably different and stronger object, not a second version of Standard.
Premium content differentiated. "Everything in Standard, plus:" header makes the upgrade relationship explicit. Delta chips ("up from 8", "hands-off") make the upgrade's value scannable without reading the description. Every Premium feature is a visible gain relative to Standard.
Screen 2 condensed. Removed the full "What you get" block (hero + timeline already convey the value) and the three-row "You're in control" block (moved to the trial-start email, where there's space to do it justice). The page now fits on one screen with the CTA visible. Timeline is the one thing we keep because it directly disarms the "I'll get silently charged" fear.
Fine print line. One line, centered, above the CTA. "Cancel anytime in one click. Payment handled by Stripe." Covers the two highest-priority trust questions without a full card block. Hyperlinkable to a details page if needed.
Secondary CTA renamed. "Just let me preview the brief" dropped. New copy is "Not now, maybe later" which matches what we actually know about the user: they are not committing to anything right now. Routes to the existing Free home (BTC only, rest gated) per your call.
Confirmed decisions (from Q&A)
Day 8 default plan is Premium. Day-5 email can mention a Standard switch, not prominent, with the tradeoff spelled out (fewer tracked assets, manual approval). Trial is 7 days. One trial per email. Fallthrough routes to the existing Free home page, not a new preview surface.
